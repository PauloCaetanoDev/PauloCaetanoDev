# Olá, eu sou o Paulo Caetano 👋

Desenvolvedor Back-End com atuação em Java, integração de sistemas e gestão de ETL. Trabalho na [Metaprime Sistemas](https://metaprime.com.br), onde além de desenvolver, lidero o departamento de migração de dados — garantindo que cada cliente tenha uma transição segura, íntegra e eficiente para o sistema.

Minha trajetória começou longe do código — passei pela gestão de estoque e pelo suporte técnico antes de migrar de vez para o desenvolvimento. Esse caminho me deu uma visão diferenciada: entendo o negócio, o usuário e o sistema ao mesmo tempo.
